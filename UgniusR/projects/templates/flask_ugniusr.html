Purpose

The purpose of the site is to have an open-ended platform to experiment and showcase/document new projects. The full source code can be found through the link on the bottom of this page.

Technology

The site is built using the Flask Framework and deployed on Ubuntu using uWSGI and Nginx.

Javascript Framework?

While I acknowledge the existence and importance of the modern Javascript options (Angular/Node.js), my priority has never been front-end development. For the purpose of this project, the current technology is sufficient and allows me to become familiar with the fundamentals.

I find it is the best use of my time to expand my knowledge of my current interests. If someday my interests shift to Javascript frameworks and front-end development, this website would very likely be the target of experimentation.

CSS

The site uses the Bootstrap library and is based on an example page provided by Bootstrap.

Structure

This project is structured to allow for future expansion by using modular components called Blueprints. There are currently three primary Blueprints: 'main', 'projects', and 'api'.

API

The API provides some resources that are used throughout the program. The API can be access through '/api/public'. In the future, I intend to expand this API to include authentication features. The resources the API provides are not very useful outside of the context of this site, but developing it provides some exposure to REST concepts.
